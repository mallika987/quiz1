# Wiki Quiz Generator

## Objective
Build a web-based system that generates quiz questions from Wikipedia articles.

## Tech Stack
- Python 3.7
- FastAPI
- HTML

## Features
- Accepts Wikipedia URL
- Generates quiz questions
- Displays history and details view
- JSON-based output

## Folder Structure
backend/, frontend/, sample_data/, screenshots/

## Run Backend
cd backend
uvicorn main:app --reload

## Sample URLs Tested
- https://en.wikipedia.org/wiki/Alan_Turing
- https://en.wikipedia.org/wiki/Ada_Lovelace
- https://en.wikipedia.org/wiki/Artificial_intelligence
